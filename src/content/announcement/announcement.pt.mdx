## 🚲 Cidade para bicicletas orientada por dados - Workshop sobre a Plataforma de Dados ATRAI

**Como os dados de mobilidade e ambientais podem contribuir para a melhoria da infraestrutura cicloviária?**  

Como parte deste workshop, apresentaremos os resultados atuais do projeto ATRAI-Bikes, financiado pelo BMFTR.  

Após uma revisão dos últimos desenvolvimentos e funcionalidades da senseBox:bike, bem como dos primeiros workshops e campanhas realizados em Münster, São Paulo, Wiesbaden, Arnsberg e Greifswald, o workshop terá como foco a **plataforma de dados de código aberto** desenvolvida no **projeto ATRAI** — uma ferramenta para coleta, visualização e análise de dados de tráfego, projetada para oferecer novas possibilidades de análise de dados relacionados à bicicleta para cidadãos, pesquisadores e administração pública.  

A plataforma utiliza, entre outras fontes, dados da **senseBox:bike**, uma plataforma móvel de sensores para registrar informações sobre temperatura, umidade, partículas em suspensão, ultrapassagens perigosas de carros (baseadas em IA), distâncias de ultrapassagem e condições do pavimento (baseadas em IA).  

O ponto central do workshop será um teste de uso da plataforma, explorando como trabalhar com os dados coletados pelas senseBox:bikes, bem como uma discussão sobre necessidades e desejos em relação a funções, interfaces, formatos de arquivo ou opções de exportação.  

**Como podemos obter insights a partir disso — e como eles podem ser usados para ajudar a projetar uma cidade mais sustentável para bicicletas?**

### O que são os Lightning Talks?

Durante o workshop, convidamos os participantes a fazer pequenas apresentações de 5-10 minutos, conhecidas como **Lightning Talks**. Elas são uma ótima oportunidade para compartilhar:

- Seu projeto ou iniciativa relacionada ao tema  
- Um desafio relacionado ao desenvolvimento urbano orientado por dados  
- Uma nova ideia ou perspectiva sobre mobilidade, qualidade do ar ou ciência cidadã  

É informal e aberto a todos - basta indicar no momento da inscrição se deseja participar!
